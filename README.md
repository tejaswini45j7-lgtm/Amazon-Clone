# Amazon Homepage Clone

A responsive, pixel-perfect clone of the Amazon homepage built from scratch using semantic HTML5 and custom CSS3. This project demonstrates foundational front-end web development concepts, including CSS Flexbox, structural component layouts, interactive hover states, and iconography integration.

---

## 🚀 Features

* **Amazon-Style Navbar:** Features a multi-element navigation bar housing a logo container, delivery address tracker, wide search input with custom selection fields, account status, returns tracking, and a dedicated shopping cart layout.
* **Secondary Navigation Panel:** A contextual sub-header highlighting product categories ("Today's Deals", "Customer Service", "Gift Cards") accompanied by an interactive hamburger menu.
* **Hero Banner Section:** A prominent background image banner containing a floating structural announcement box that matches the actual Amazon interface.
* **Product Category Grid:** A flexible multi-row display showcasing distinct category cards (Clothes, Electronics, Pet Care, etc.) with clean typography and image spaces.
* **Multi-tiered Footer:** A comprehensive multi-stage footer design housing a "Back to Top" navigation anchor, variable link lists, and a bottom copyright notice layout.

---

## 🛠️ Tech Stack

* **HTML5:** Semantic architecture layout (`<header>`, `<nav>`, `<ul>`, `<footer>`).
* **CSS3:** Custom styles utilizing **CSS Flexbox**, explicit alignments, background images, absolute sizing, and smooth dynamic transitions.
* **FontAwesome:** Scalable UI vector icons (location pin, magnifying search glass, cart tracking icon, and side-nav bars).
